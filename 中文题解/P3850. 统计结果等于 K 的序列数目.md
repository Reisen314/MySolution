3850. 统计结果等于 K 的序列数目

[原题](https://leetcode.cn/problems/count-sequences-to-k/description/)

## 思路
既然是“有理除法”，考虑分解质因数，把各质因数指数的系数用字典存储，e.g. 21 ->{3:1,7:1}
我做的时候想象成k是由某些积木拼成，然后nums里存储了n堆积木，然后按需去nums里找积木（，虽然这样想解释不了除法，总归能简化一下问题


## 解题过程
具体做的时候，想办法把走到k步时，可以达到的数都存储起来，最后查找就可以，为了可以存进字典，把状态量整理成元组

e.g.{3:1,7:1} -> (1,1)表示 3 * 7 = 21 (1,0) 表示 3*1=3

## 复杂度

时间复杂度: $$O(3^n)$$ 

空间复杂度: $$O(n)$$




``` python
from collections import defaultdict
from math import isqrt
#分解质因数，存储成字典,e.g. 24 -> {2:3,3:1}
def factorize(x):
    d=dict()
    tmp=x
    for i in range(2,isqrt(tmp)+1):
        while x%i == 0:
            d[i] = d.get(i,0) + 1
            x//=i
    if x>1:
        d[x]=d.get(x,0) + 1
    return d
class Solution:  
    def countSequences(self, nums, k: int) -> int:
        #对k分解质因数
        fac=factorize(k)
        #找nums里所有的质因数 e.g.[2,3,4,7] -> [2,3,7]
        temp=[factorize(x).keys() for x in nums]
        primes=[]
        for x in temp:
            for k in x:
                primes.append(k)
        prime_set=set(primes)
        primes=list(prime_set)
        #如果k有nums中没有的质因数，直接退出e.g.nums=[2,3],k=7,不会有解
        for y in fac.keys():
            if not y in prime_set:
                return 0
        #把k的质因数存成tuple
        fac_cnt=[]
        for p in primes:
            try:
                fac_cnt.append(fac[p])
            except:
                fac_cnt.append(0)
        fac_cnt=tuple(fac_cnt)
        #初始化dp
        dp=defaultdict(int)
        dp[tuple([0]*len(primes))]+=1

        for num in nums:

            num_fac = factorize(num)
        
            delta = [num_fac.get(p, 0) for p in primes]
        
            new_dp = defaultdict(int)
        
            for state, count in dp.items():
                #更新状态
                #1.不变
                new_dp[state]+=count
                #2.乘num
                add_state=tuple(state[i]+delta[i] for i in range(len(primes)))
                new_dp[add_state]+=count
                #3.除num
                sub_state=tuple(state[i]-delta[i] for i in range(len(primes)))
                new_dp[sub_state]+=count
            #更新
            dp = new_dp
            print(dp)
        return dp[fac_cnt]



```
